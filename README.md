# Calistrack – 100‑Day Calisthenics Journey

**Calistrack** is a Python command‑line program that helps you stay consistent for 100 days. It gives you daily workouts, tracks your skill progress (pull‑ups, handstands, etc.), and suggests protein intake based on your weight.

## Features
- 100‑day progressive workout plan (easy → hard)
- Log workouts and automatically improve skill levels
- Protein goal calculator + searchable food database
- Visual calendar with emoji progress
- All data saved locally as JSON (no internet needed)

## How to Run
```bash
python main.py
